## Install

```bash
npm install --save react-truncated-component
```

## Usage

```javascript
function TruncatedText() {
  return (
    <ReactTruncatedComponent ellipsis="..." numberOfLines={5} lineHeight={23}>
      <p>{/*


    put your long text right here


    */}</p>
    </ReactTruncatedComponent>
  );
}
```
